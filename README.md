# Emotion Detector - AI Web Application

This repository contains the code for the **Emotion Detector** web application. It uses the **IBM Watson NLP library** to analyze text and detect emotions: anger, disgust, fear, joy, and sadness, along with identifying the dominant emotion.

## Features
- Interacts with Watson NLP `EmotionPredict` API.
- Implements robust error handling and fallback mechanism.
- Web deployment via Flask server.
- 100% test coverage with Python unit tests.
- Perfect 10/10 pylint score for static code analysis.
